
Install-ready clean distribution for a multi-language newsroom platform inspired by BBC, Sky News, and Al Jazeera.

- Sanitized environment template
- Installer ready for fresh setup
- Unified version badge
- Clean branding storage
- Admin guard regex fix
- Documentation starter for V3 newsroom architecture

`v3.5.0-stable`

1. Upload files to your hosting.
2. Copy `.env.example` to `.env`.
3. Fill database and site settings.
4. Open `install.php`.

- Change admin credentials immediately after install.
- Delete or lock installer after installation.
- Ensure `storage/` and upload folders are writable.
