# اقتراحات إضافية بعد Skeleton v4

## 1) Database bridge
إنشاء طبقة repository حقيقية تستخدم الجداول الحالية بدل البيانات التجريبية.

## 2) Redirect map
ملف أو جدول يجمع كل التحويلات القديمة إلى canonical routes.

## 3) Menu service
توحيد الهيدر والفوتر والقوائم المحمولة عبر خدمة واحدة.

## 4) Theme customizer
إعدادات ألوان وشعارات وروابط اجتماعية من لوحة التحكم.

## 5) SEO center
خدمة مركزية للـ meta/canonical/schema/og/twitter.

## 6) Media library
إدارة صور مصغرات و WebP وتنظيف الملفات اليتيمة.

## 7) Search & autosuggest
بحث موحد للأخبار والصفحات والتصنيفات مع اقتراحات.

## 8) Editorial workflow
Draft / Review / Published / Scheduled.

## 9) Health check
صفحة فحص للتخزين والكاش والروابط وقاعدة البيانات.

## 10) Plugin hooks
أحداث مثل: `news.published`, `page.updated`, `theme.rendered`.
