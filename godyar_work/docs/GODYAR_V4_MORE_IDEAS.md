# اقتراحات إضافية لـ Godyar CMS v4

## إضافات تقنية قوية
- Redirect Manager داخل لوحة التحكم مع عداد hits.
- Menu Builder موحد للهيدر والفوتر والموبايل.
- Media Library مع WebP/thumbnail/orphan cleanup.
- Search autosuggest مع cache.
- Health Check dashboard لفحص المسارات والكاش وقاعدة البيانات.
- Settings snapshots واسترجاع سريع للإعدادات.

## إضافات تحريرية
- نظام Revisions للمقالات والصفحات.
- Scheduled publishing.
- Editor review workflow.
- صفحات كتّاب Author profiles.
- Tags + archives.
- Breaking news ticker و Trending block.

## تحسينات تجربة المستخدم
- Dark mode.
- Font-size switcher.
- Article reading time.
- Sticky smart header.
- Skeleton loaders.
- Accessibility mode.

## تحسينات SEO
- Breadcrumb schema.
- Dynamic XML sitemaps متعددة.
- Canonical manager.
- Open Graph defaults من الإعدادات.
- Redirect chain checker.

## أولويات التنفيذ المقترحة
1. MenuService + RedirectRepository
2. Contact persistence + anti-spam
3. SitemapService + SeoService expansion
4. MediaService + image variants
5. Admin redirect manager + health page
